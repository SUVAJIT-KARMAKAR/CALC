## RUN THE PROJECT LOCALLY

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd calculator-application
```

Installing dependencies

```bash
  npm install
```

Start the application

```bash
  npm run compile-scss
```
